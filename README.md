# 📊 Mobile Sales Analysis Dashboard – Power BI

An interactive Power BI dashboard designed to analyze mobile sales performance, customer behavior, and transaction trends through dynamic visualizations and KPI reports.

## 🚀 Key Highlights

* Analyzed total sales, quantity, transactions, and customer ratings
* Created interactive charts for city-wise and product-wise performance
* Added dynamic filters and slicers for better business analysis
* Visualized payment methods and monthly sales trends

## 🛠 Technologies Used

* Power BI
* SQL
* Excel
* Data Analysis & Visualization
